# 利用方法

1. 投稿閲覧（ログインせずに閲覧できます）
2. 新規ユーザー登録

   - メールアドレスで登録
   - Google で登録

3. ログイン
4. 投稿の作成
5. 投稿の編集/削除
6. 投稿へのコメント

<br>
